# ProxyPool
## 直接使用

* 自动检测当前目录下是否存在命名为`Proxy.csv`的文件
* 检测到则提示是否重新获取，未检测到则自动获取代理信息，
* 信息获取结束后（或不重新获取）提示是否更新代理信息
* __默认`timeout==3`，可能出现由于网络原因代理大量无效的情况，更新后提示检测数，通过数以及是否修改代理信息文件__
